## Introduction
Ruby on Rails is an MVC framework based on the programming language ruby.
It helps you quickly create APIs and server-side web apps. The default view
on the server-side is served by templating engines like ERB, HAML, and Slim.

But when it comes to using the JavaScript frameworks like ReactJS for the view
instead of Templating engines, things become a little confusing on connecting
rails with react.

This post will guide you through 3 ways to work with react and rails.
- React-Rails tutorial with JSRoutes
- Monolithic web app with react-rails
- Separate rails API and react client app
